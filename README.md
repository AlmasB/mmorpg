Orion MMORPG
======

Final year project, an mmorpg game in java.

## Compile, Run ##

Required:
- Oracle JDK 1.8+

other dependencies will be handled by maven

Commands:

(in project root)

```bash
$ mvn package
$ mvn exec:java
```
